# Studio S.Path

Website for Studio S.Path, an architecture and interior design studio based in Pune, India.

## Overview

A modern, responsive website showcasing the studio's architectural work and practice. The site emphasizes spatial sequences, material honesty, and thoughtful design through a calm, architectural interface.

## Features

- **Intro Animation**: Elegant logo reveal animation on first visit
- **Project Showcase**: Detailed project pages with parallax effects and image sequences
- **Responsive Design**: Optimized for all screen sizes, including large desktop displays (up to 30" monitors)
- **Smooth Animations**: Subtle parallax effects and scroll-based animations using GSAP
- **Clean Typography**: Architectural, minimal design language throughout

## Pages

- **Landing**: Hero section with philosophy statement and selected works preview
- **Projects**: Filterable project listing with typology-based sorting
- **Project Detail**: Comprehensive project pages with story sequences, constraints, and responses
- **About**: Studio philosophy, practice information, and team details
- **Contact**: Contact information and inquiry form

## Technology

Built with React, Vite, Tailwind CSS, and GSAP for animations.
